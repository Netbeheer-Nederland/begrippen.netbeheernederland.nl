# TTL-bestanden

TTL-bestanden conformeren zich aan [NL-SBB](https://docs.geostandaarden.nl/nl-sbb/nl-sbb/). Het begrippenkader is opgesplitst in meerdere TTL-bestanden:

- één bestand voor het kader (`00-kader.ttl`)
- één bestand voor bronnen (`01-bronnen.ttl`)
- één bestand per domein (bv. `C1-klant.ttl`)
